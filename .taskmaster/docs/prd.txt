# GainDeuk - 암호화폐 초심자 친화적 AI 추천 서비스

## 프로젝트 개요
GainDeuk은 암호화폐 초심자를 위한 AI 추천 서비스입니다. 복잡한 차트나 지표 대신 "왜 이 코인을 추천하는지"를 쉽게 설명하고, 사용자의 투자 성향에 맞는 맞춤형 추천을 제공합니다.

## 핵심 목표
- 초심자도 쉽게 이해할 수 있는 직관적인 인터페이스
- AI 기반 실시간 매매 가이드 제공
- 개인화된 투자 성향 분석 및 추천
- 복잡한 분석은 AI가 처리하고 사용자는 신호만 따라하는 구조

## 주요 기능

### 1. 메인 대시보드
- AI 추천 코인 3-5개 표시
- 추천 이유 간단한 3줄 설명
- 예상 수익률 퍼센트로 표시
- 위험도 1-5단계 색상 구분
- 사용자 투자 성향 분석 결과

### 2. 실시간 매매 가이드 (핵심 기능)
- "지금 사세요/팔세요" 실시간 신호
- AI가 자동으로 계산한 손절가, 목표가, 포지션 크기
- 스캘핑/스윙/데이트레이딩 전략별 가이드
- 단계별 매매 체크리스트
- 매매 신호 근거 (뉴스, 기술분석, 고래활동, 감정분석)
- 유효시간 카운트다운
- 매매 실행 기능

### 3. 내 성향 분석
- 투자 성향 테스트 (5-10개 질문)
- AI 분석 결과 및 성향별 설명
- 맞춤 추천 설정
- 투자 전략 추천

### 4. 코인 목록
- 전체 코인 간단한 카드 형태
- 검색 기능
- 상세 정보 모달
- AI 추천 코인 하이라이트

### 5. 설정
- 알림 설정
- 언어 설정
- 테마 설정

## 기술 스택
- Frontend: React 18 + TypeScript + Vite
- Styling: Styled Components
- State Management: React Query
- Routing: React Router DOM
- Icons: Lucide React
- Notifications: React Hot Toast
- HTTP Client: Axios

## 디자인 원칙
- 초심자 친화적: 복잡한 차트나 지표 최소화
- 직관적: 한글로 된 쉬운 설명, 직관적인 아이콘과 색상
- 근거 중심: "왜 이 코인인가?" 명확한 이유 제시
- 단계별 학습: 초심자 → 중급자 → 고급자 단계별 정보 제공

## API 연동
- 백엔드 서버: http://localhost:3000
- Swagger API 문서: http://localhost:3000/api-docs
- 주요 엔드포인트:
  - /api/dashboard - 메인 대시보드 데이터
  - /api/recommendations - AI 추천 코인
  - /api/trading/signals - 실시간 매매 신호
  - /api/trading/execute - 매매 실행
  - /api/analyze-profile - 사용자 성향 분석
  - /api/coins - 코인 정보

## 개발 우선순위
1. 프로젝트 초기 설정 및 기본 구조
2. 실시간 매매 가이드 페이지 (최우선)
3. TradingSignalCard 컴포넌트
4. AI 추천 카드 컴포넌트
5. 사용자 성향 분석 페이지
6. API 연동 및 실시간 업데이트
7. 반응형 디자인 및 최적화

## 핵심 차별화 포인트
- "전문가가 옆에 있는 느낌"의 실시간 매매 가이드
- AI가 모든 것을 자동 계산 (손절가, 목표가, 포지션 크기)
- 초심자도 바로 따라할 수 있는 단계별 가이드
- 실시간 업데이트 (10초마다 매매 신호 새로고침)
- 개인화된 매매 전략

## 성공 지표
- 사용자가 AI 신호를 따라 매매하는 비율
- 초심자의 이해도 및 만족도
- 실시간 매매 가이드 사용률
- 사용자 재방문율
